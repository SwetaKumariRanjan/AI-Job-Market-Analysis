# AI-Job-Market-Analysis
End-to-end Data Analyst project using Python, SQL, and Power BI to analyze AI job market trends, salaries, and skills demand.
This project analyzes the global AI job market using real-world datasets. It focuses on job trends, salary insights, and in-demand skills.

Tools Used
Python (Pandas, NumPy)
SQL (MySQL)
Power BI
Excel

Key Insights
Python and SQL are the most in-demand skills
Senior roles offer higher salaries
Remote jobs are increasing globally
USA leads in AI job opportunities

Project Structure
Data Cleaning: Python
Analysis: SQL
Dashboard: Power BI

Dashboard Features
Salary Insights Dashboard
AI Job Market Analysis
Skills Demand Analysis
